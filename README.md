# 🔒 Cybersecurity Awareness Chatbot

A comprehensive C# console application that serves as an interactive cybersecurity awareness assistant with task management, quiz functionality, and natural language processing capabilities.

## 📋 Features

### 1. Task Assistant
- Add cybersecurity-related tasks
- Set reminders with natural language (e.g., "tomorrow", "in 3 days")
- View and manage tasks
- Database storage support

### 2. Cybersecurity Quiz
- 10+ multiple-choice questions
- Immediate feedback with explanations
- Score tracking with personalized feedback
- Covers phishing, passwords, 2FA, and more

### 3. Natural Language Processing
- Keyword detection for cybersecurity topics
- Sentiment analysis (worried, frustrated, curious)
- Memory and recall (name, preferences)
- Regular expression pattern matching

### 4. Activity Log
- Tracks all user interactions
- Recent activity display
- Full history available
- Timestamp recording

### 5. Voice Integration
- Welcome voice greeting
- Audio file support
